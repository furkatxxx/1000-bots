## О проекте
**1000 ботов** — система-конвейер для поиска и анализа бизнес-идей через AI-агентов.

Большое видение: управление тысячами бизнесов без сотрудников, только AI-агенты.
Сейчас: первое звено — агенты-разведчики (сбор трендов → генерация идей).

## Стек
- Next.js 16 + TypeScript + Tailwind CSS 4
- Prisma 6 + SQLite
- Claude API (Haiku) для генерации идей
- Порт: 1000

## Запуск
```bash
export PATH="$HOME/.local/node/bin:$PATH"
npm run dev
```
Открыть: http://localhost:1000

## Структура
- `src/app/` — страницы и API-роуты
- `src/lib/` — бизнес-логика (db, ai-brain, collectors)
- `src/hooks/` — хуки для API-запросов
- `src/components/` — UI-компоненты
- `prisma/` — схема БД

## Правила
- Дизайн в стиле Apple (CSS-переменные, скруглённые углы, тени)
- React.memo() на всех UI-компонентах
- Лимиты: UI < 200 строк, роуты < 300, сервисы < 400
- Не создавай файлы "на будущее"
- Проверяй `npm run build` перед заявлением о готовности
