## Общая картина

**Текущее состояние (ВСЕ ЭТАПЫ ЗАВЕРШЕНЫ):**
```
Тренды (8 источников) → AI-мозг → Идеи → Экспертный совет (4 специалиста + автовалидация) → Deep Dive → Telegram ТОП-5
                                             ↓
                                   Реальные данные: Вордстат + DaData + ЕГРЮЛ
```

## Архитектура сбора данных

```
8 коллекторов трендов:
├── Hacker News (бесплатно)
├── Google Trends (бесплатно)
├── NewsAPI (нужен ключ)
├── GitHub Trending (бесплатно)
├── Product Hunt (бесплатно)
├── Яндекс Вордстат (нужен OAuth-токен) ✅
├── Telemetr.io (нужен API-ключ) ⏳
└── VK Тренды (нужен сервисный токен) ⏳

3 валидатора (для экспертного совета):
├── Вордстат → поисковый спрос ✅
├── DaData → компании в нише (ОКВЭД) ⏳
└── ЕГРЮЛ → финансы конкурентов ✅
```
